Raspamento Básico da Web em Java

A primeira coisa que você precisa aprender é como raspar um site estático em Java. Você pode pensar em um site estático como uma coleção de documentos HTML pré-construídos. Cada uma dessas páginas HTML terá seus arquivos CSS e JavaScript. Um site estático depende da renderização do lado do servidor.

Em uma página estática, o conteúdo é incorporado no documento HTML fornecido pelo servidor. Então, você não precisa de um navegador para extrair dados dele. Em detalhes, a raspagem estática da web é sobre:

  1. Baixar uma página web
  2. Analisando o documento HTML recuperado do servidor.
  3. Selecionar os elementos HTML contendo os dados de interesse da página web.
  4. Extraindo os dados a partir deles

Em Java, raspar páginas da web não é difícil, especialmente quando se trata de web scraping estático. Vamos agora aprender o básico de raspagem de web com Java.

Vamos agora recuperar a lista de todos os links de paginação para raspar todo o site. É disso que se trata o web crawling. Clique com o botão direito no botão do número de paginação do elemento HTML e escolha a opção "Inspecionar".

Se você quiser raspar todas as páginas da web, você tem que implementar alguma lógica de rastreamento. Além disso, você precisará confiar em algumas listas e conjuntos para evitar raspar uma página da web duas vezes. Você pode implementar a lógica de rastreamento da web para visitar uma quantidade limitada de páginas da Web controladas por limitComo se segue:

Scraper.java (taxa)
package com.zenrows; 
 
import com.zenrows.data.Product; 
import org.jsoup.*; 
import org.jsoup.nodes.*; 
import org.jsoup.select.*; 
import java.io.IOException; 
import java.util.*; 
 
public class Scraper { 
	public static void scrapeProductPage( 
			List<Product> products, 
			Set<String> pagesDiscovered, 
			List<String> pagesToScrape 
	) { 
		// the current web page is about to be scraped and 
		// should no longer be part of the scraping queue 
		String url = pagesToScrape.remove(0); 
 
		pagesDiscovered.add(url); 
 
		// ... scraping logic omitted for brevity

		Elements paginationElements = doc.select("a.page-numbers");
 
		// iterating over the pagination HTML elements 
		for (Element pageElement : paginationElements) { 
			// the new link discovered 
			String pageUrl = pageElement.attr("href"); 
 
			// if the web page discovered is new and should be scraped 
			if (!pagesDiscovered.contains(pageUrl) && !pagesToScrape.contains(pageUrl)) { 
				pagesToScrape.add(pageUrl); 
			} 
 
			// adding the link just discovered 
			// to the set of pages discovered so far 
			pagesDiscovered.add(pageUrl); 
		} 
	} 
 
	public static void main(String[] args) { 
		// initializing the list of Java object to store 
		// the scraped data 
		List<Product> products = new ArrayList<>(); 
 
		// initializing the set of web page urls 
		// discovered while crawling the target website 
		Set<String> pagesDiscovered = new HashSet<>(); 
 
		// initializing the queue of urls to scrape 
		List<String> pagesToScrape = new ArrayList<>(); 
		// initializing the scraping queue with the 
		// first pagination page 
		pagesToScrape.add("https://www.scrapingcourse.com/ecommerce/page/1/"); 
 
		// the number of iteration executed 
		int i = 0; 
		// to limit the number to scrape to 5 
		int limit = 12; 
 
		while (!pagesToScrape.isEmpty() && i < limit) { 
			scrapeProductPage(products, pagesDiscovered, pagesToScrape); 
			// incrementing the iteration number 
			i++; 
		} 
 
		System.out.println(products.size()); 
 
		// writing the scraped data to a db or export it to a file... 
	} 
}

scrapeProductPage()raspa uma página da Web, descobre novos links para raspar e adiciona sua URL à fila de raspagem. Você já definiu o limitpara 12 porque o site tem 12 páginas. Então, no final do whileciclo, pagesToScrapeVai ficar vazio e pagesDiscoveredconterá todos os 12 URLs de paginação.

Aqui está o código completo:

Scraper.java (taxa)
package com.zenrows;

import com.zenrows.data.Product;
import org.jsoup.*;
import org.jsoup.nodes.*;
import org.jsoup.select.*;
import java.io.IOException;
import java.util.*;
import java.util.concurrent.ExecutorService;
import java.util.concurrent.Executors;
import java.util.concurrent.TimeUnit;

public class Scraper {
    public static void scrapeProductPage(
            List<Product> products,
            Set<String> pagesDiscovered,
            List<String> pagesToScrape
    ) {
        if (!pagesToScrape.isEmpty()) {
            // the current web page is about to be scraped and
            // should no longer be part of the scraping queue
            String url = pagesToScrape.remove(0);

            pagesDiscovered.add(url);

            // initializing the HTML Document page variable
            Document doc;

            try {
                // fetching the target website
                doc = Jsoup
                        .connect(url)
                        .userAgent("Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36")
                        .get();
            } catch (IOException e) {
                throw new RuntimeException(e);
            }

            // retrieving the list of product HTML elements
            // in the target page
            Elements productElements = doc.select("li.product");

            // iterating over the list of HTML products
            for (Element productElement : productElements) {
                Product Product = new Product();

                // extracting the data of interest from the product HTML element
                // and storing it in Product
                Product.setUrl(productElement.selectFirst("a").attr("href"));
                Product.setImage(productElement.selectFirst("img").attr("src"));
                Product.setName(productElement.selectFirst("h2").text());
                Product.setPrice(productElement.selectFirst("span").text());

                // adding Product to the list of the scraped products
                products.add(Product);
            }

            // retrieving the list of pagination HTML element
            Elements paginationElements = doc.select("a.page-numbers");

            // iterating over the pagination HTML elements
            for (Element pageElement : paginationElements) {
                // the new link discovered
                String pageUrl = pageElement.attr("href");

                // if the web page discovered is new and should be scraped
                if (!pagesDiscovered.contains(pageUrl) && !pagesToScrape.contains(pageUrl)) {
                    pagesToScrape.add(pageUrl);
                }

                // adding the link just discovered
                // to the set of pages discovered so far
                pagesDiscovered.add(pageUrl);
            }

            // logging the end of the scraping operation
            System.out.println(url + " -> page scraped");
        }
    }

    public static void main(String[] args) { 
		// initializing the list of Java object to store 
		// the scraped data 
		List<Product> products = new ArrayList<>(); 
 
		// initializing the set of web page urls 
		// discovered while crawling the target website 
		Set<String> pagesDiscovered = new HashSet<>(); 
 
		// initializing the queue of urls to scrape 
		List<String> pagesToScrape = new ArrayList<>(); 
		// initializing the scraping queue with the 
		// first pagination page 
		pagesToScrape.add("https://www.scrapingcourse.com/ecommerce/page/1/"); 
 
		// the number of iteration executed 
		int i = 0; 
		// to limit the number to scrape to 5 
		int limit = 12; 
 
		while (!pagesToScrape.isEmpty() && i < limit) { 
			scrapeProductPage(products, pagesDiscovered, pagesToScrape); 
			// incrementing the iteration number 
			i++; 
		} 
 
		System.out.println(products.size());
        // writing the scraped data to a db or export it to a file...
    }
}

Scraper.java (taxa)
package com.zenrows;

import com.zenrows.data.Product;
import org.jsoup.*;
import org.jsoup.nodes.*;
import org.jsoup.select.*;
import java.io.IOException;
import java.util.*;
import java.util.concurrent.ExecutorService;
import java.util.concurrent.Executors;
import java.util.concurrent.TimeUnit;

public class Scraper {
    public static void scrapeProductPage(
            List<Product> products,
            Set<String> pagesDiscovered,
            List<String> pagesToScrape
    ) {
        if (!pagesToScrape.isEmpty()) {
            // the current web page is about to be scraped and
            // should no longer be part of the scraping queue
            String url = pagesToScrape.remove(0);

            pagesDiscovered.add(url);

            // initializing the HTML Document page variable
            Document doc;

            try {
                // fetching the target website
                doc = Jsoup
                        .connect(url)
                        .userAgent("Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36")
                        .get();
            } catch (IOException e) {
                throw new RuntimeException(e);
            }

            // retrieving the list of product HTML elements
            // in the target page
            Elements productElements = doc.select("li.product");

            // iterating over the list of HTML products
            for (Element productElement : productElements) {
                Product Product = new Product();

                // extracting the data of interest from the product HTML element
                // and storing it in Product
                Product.setUrl(productElement.selectFirst("a").attr("href"));
                Product.setImage(productElement.selectFirst("img").attr("src"));
                Product.setName(productElement.selectFirst("h2").text());
                Product.setPrice(productElement.selectFirst("span").text());

                // adding Product to the list of the scraped products
                products.add(Product);
            }

            // retrieving the list of pagination HTML element
            Elements paginationElements = doc.select("a.page-numbers");

            // iterating over the pagination HTML elements
            for (Element pageElement : paginationElements) {
                // the new link discovered
                String pageUrl = pageElement.attr("href");

                // if the web page discovered is new and should be scraped
                if (!pagesDiscovered.contains(pageUrl) && !pagesToScrape.contains(pageUrl)) {
                    pagesToScrape.add(pageUrl);
                }

                // adding the link just discovered
                // to the set of pages discovered so far
                pagesDiscovered.add(pageUrl);
            }

            // logging the end of the scraping operation
            System.out.println(url + " -> page scraped");
        }
    }

    public static void main(String[] args) throws InterruptedException {
        // initializing the list of Java object to store
        // the scraped data
        List<Product> products = Collections.synchronizedList(new ArrayList<>());

        // initializing the set of web page urls
        // discovered while crawling the target website
        Set<String> pagesDiscovered = Collections.synchronizedSet(new HashSet<>());

        // initializing the queue of urls to scrape
        List<String> pagesToScrape = Collections.synchronizedList(new ArrayList<>());
        // initializing the scraping queue with the
        // first pagination page
        pagesToScrape.add("https://www.scrapingcourse.com/ecommerce/page/1/");

        // initializing the ExecutorService to run the
        // web scraping process in parallel on 4 pages at a time
        ExecutorService executorService = Executors.newFixedThreadPool(4) ;

        // launching the web scraping process to discover some
        // urls and take advantage of the parallelization process
        scrapeProductPage(products, pagesDiscovered, pagesToScrape);

        // the number of iteration executed
        int i = 1;
        // to limit the number to scrape to 5
        int limit = 12;

        while (!pagesToScrape.isEmpty() && i < limit) {
            // registering the web scraping task
            executorService.execute(() -> scrapeProductPage(products, pagesDiscovered, pagesToScrape));

            // adding a 200ms delay for avoid overloading the server
            TimeUnit.MILLISECONDS.sleep(200);

            // incrementing the iteration number
            i++;
        }

        // waiting up to 300 seconds to all pending tasks to end
        executorService.shutdown();
        executorService.awaitTermination(300, TimeUnit.SECONDS);

        System.out.println(products.size());

        // writing the scraped data to a db or export it to a file...
    }
}

Tenha em mente que ArrayListE a HashSetNão são thread-safe em Java. É por isso que você precisa envolver suas coleções com Collections.synchronizedList()E a Collections.synchronizedSet(), respectivamente. Esses métodos os transformarão em coleções seguras para threads que você pode usar em threads.

Então, você pode usar ExecutorServicespara executar tarefas de forma assíncrona. Graças a ExecutorServices, você pode executar e gerenciar tarefas paralelas sem esforço. Especificamente, newFixedThreadPool()permite inicializar um Executorque pode executar simultaneamente tantos threads quanto o número passado para o método de inicialização.

Você não quer sobrecarregar o servidor de destino ou a máquina local. É por isso que você precisa adicionar alguns milissegundos de tempo limite entre os tópicos com sleep()- A . (í a , , , , , ínte , . Seu objetivo é executar web scraping, não um ataque DOS.

Então, lembre-se sempre de desligar o seu ExecutorServicee liberar o seu recurso. Desde quando o código existe whileciclo alguma tarefa ainda pode estar em execução, você deve usar o awaitTermination()método.

Você deve chamar esse método após uma solicitação de desligamento. Em detalhe, awaitTermination()bloqueia o código e aguardar que todas as tarefas sejam concluídas dentro do intervalo de tempo passado como parâmetro.

Raspando sites de conteúdo dinâmico em Java

Não se esqueça que uma página da web é mais do que o documento HTML correspondente. As páginas da Web podem realizar solicitações HTTP no navegador via AJAX. Este mecanismo permite que as páginas da Web recuperem dados de forma assíncrona e atualizem o conteúdo mostrado ao usuário de acordo.

A maioria dos sites agora depende de solicitações de API de frontend para recuperar dados. Essas solicitações são chamadas AJAX. Portanto, essas chamadas de API fornecem dados valiosos que você não pode ignorar quando se trata de raspagem da web. Você pode cheirar essas chamadas, replicá-las em seu script de raspagem e recuperar esses dados.

Para cheirar uma chamada AJAX, use o DevTools seu navegador. Clique com o botão direito do mouse em uma página da Web, escolha "Inspecionar" e selecione a guia "Rede". Na guia "Fetch/XHR", você encontrará a lista de chamadas AJAX que a página da Web é executada, conforme abaixo.

Aqui, você pode recuperar todas as informações necessárias para replicar essas chamadas em seu script de raspagem da web. No entanto, esta não é a melhor abordagem.

Raspagem da Web com um navegador sem cabeça

As páginas da Web realizam a maioria das chamadas AJAX em resposta à interação do usuário. É por isso que você precisa de uma ferramenta para carregar uma página da Web em um navegador e replicar a interação do usuário. É disso que se trata um navegador sem cabeça.

Em detalhes, um navegador sem cabeça é um navegador da Web sem interface gráfica que permite controlar programaticamente uma página da web. Em outros termos, um navegador sem cabeça permite que você instrua um navegador da Web para executar algumas tarefas.

Graças a um navegador sem cabeça, você pode interagir com uma página da web através de JavaScript como um ser humano faria. Uma das bibliotecas mais populares em Java, oferecendo funcionalidade de navegador sem cabeça é o Selenium WebDriver.
